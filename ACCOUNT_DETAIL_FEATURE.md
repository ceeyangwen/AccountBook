# 账号详情页面功能说明

## 新增功能

### 账号详情页面
在点击账号列表中的任意账号时，会跳转到账号详情页面，展示该账号的所有操作记录。

## 主要功能

### 1. 账号基本信息展示
- 显示账号图标、名称和分类
- 显示当前余额
- 对于负债/信用类账户，余额显示为红色

### 2. 操作统计
- 支出总额：统计该账号的所有支出记录
- 收入总额：统计该账号的所有收入记录
- 转出总额：统计从该账号转出的金额
- 转入总额：统计转入该账号的金额

### 3. 操作记录列表
- 按日期分组显示所有操作记录
- 支持查看以下类型的记录：
  - 支出记录：显示支出金额和类别
  - 收入记录：显示收入金额和类别
  - 转账记录：显示转账金额、转出/转入的对方账号
- 每条记录显示备注信息

### 4. 记录操作
- 点击记录可跳转到编辑页面修改记录
- 可直接删除无效记录

## 使用方法

1. 进入「账户」页面
2. 点击任意账号卡片进入详情页面
3. 查看该账号的所有操作记录和统计信息
4. 点击记录可编辑或删除
